    """Log message to console_log.txt with timestamp"""
    # 파일이 닫혔는지 확인 후 쓰기 시도  ← 추가
    if _console_log_file.closed:

        _console_log_file.write(f"[{timestamp}] {message}\n")

## 수정 내용

1. **`_console_log_file.closed` 체크**: 파일이 이미 닫혔으면 즉시 리턴
2. **`try-except` 추가**: 혹시 체크와 쓰기 사이에 race condition이 발생하더라도 `ValueError`를 안전하게 무시

이제 `finally` 블록에서 파일을 닫은 후에도 `log_message()`가 호출되더라도 에러가 발생하지 않습니다.
